# 🚧 RoadGuard AI

## 📌 Problem Statement

Road accidents caused by potholes are a major issue. Manual road inspection is slow and expensive.

RoadGuard AI automatically detects potholes using Computer Vision and Deep Learning.

---

## 🚀 Features

- Real-time pothole detection
- Custom YOLOv8 model
- Voice alert
- Detection history
- CSV logging
- Streamlit Dashboard
- Automatic image saving

---

## 🛠 Tech Stack

- Python
- YOLOv8
- OpenCV
- Streamlit
- Pandas
- Ultralytics
